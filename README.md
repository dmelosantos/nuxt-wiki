Sample project implementing a basic Wiki with NuxtJS and Express + Sequelize

# Backend

## Installation

Just run and you are good to go
```shell script
yarn
```

## Usage

Just run and you are good to go
```shell script
yarn start
```

For tests you can run
Just run and you are good to go
```shell script
yarn test
```

## Tech Stack
-NodeJS
-Express
-Sequelize

# Frontend

## Installation

Just run and you are good to go
```shell script
yarn
```

## Usage

This project is pre-configured to use the backend on http://localhost:8081.

Run the backend first before running this project

For development mode run:

```shell script
yarn dev
```

To build for distribution

```shell script
yarn build
```

For production mode

```shell script
yarn start
```

## Tech Stack
-NuxtJS
-Vue
-Vuetify
